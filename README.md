# TypeScript node setup

***A TypeScript setup for node projects***


---

## Usage

```bash
# have TypeScript installed
npm install -g typescript

# update dependencies
# with npm update (old npm)
npm update

# or 
# with npm-check ($ npm install -g npm-check) =>
npm-check -u

# install dependencies
npm install

# run with nodemon
npm run dev

# compile ts
npm run build

# start js
npm start
```
